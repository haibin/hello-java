# hello-java

## No dependency

When there is no dependency, we run the app like below.

```
$ mvn clean package
$ java -cp ./target/hello-java-1.0-SNAPSHOT.jar com.mycompany.app.App
hello world
```

We can use the `maven-jar-plugin` to build a runnable jar.

```
$ mvn clean package
$ java -jar target/hello-java-1.0-SNAPSHOT.jar 
hello world
```

## With dependency

We can use `maven-shade-plugin` to generate a fat jar which includes all the dependency.

Before we use the plugin, the jar is very small.

```
$ ls -lh target/hello-java-1.0-SNAPSHOT.jar
-rw-r--r--  1 haibin  staff   3.7K Mar  2 00:01 target/hello-java-1.0-SNAPSHOT.jar
```

After we integrate the plugin, the generated jar is a lot bigger.

```
$ mvn clean package
$ ls -lh target/hello-java-1.0-SNAPSHOT.jar
-rw-r--r--  1 haibin  staff   2.7M Mar  2 00:19 target/hello-java-1.0-SNAPSHOT.jar
```

Run it.

```
java -jar target/hello-java-1.0-SNAPSHOT.jar
00:27:43.309 [main] INFO com.mycompany.app.AppLogging - in foo()
hello world logging
```
